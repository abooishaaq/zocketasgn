# zocket assigment

## build

- `make all`

## hello server

- serves "Hello, World!" on port 8080

## books server

[url](https://gobooks.abooishaaq.repl.co/)

- `GET` `/books` - get all books
- `POST` `/books` - add new book
- `POST` `/books/:id` - update book
- `DELETE` `/books/:id` - delete book
